# 使用 typescript 封装 axios

## 安装 TS 环境

1. 下载 node

[node 官网](https://nodejs.org/zh-cn/)

2. 安装 Typescript

[Typescript 官网](https://www.tslang.cn/)

## 需求分析

### Features

- 在浏览器端使用 `XMLHttpRequest` 对象通讯
- 支持 Promise API
- 支持请求和响应的拦截器
- 支持请求数据和响应数据的转换
- 支持请求的取消
- JSON 数据的自动转换
- 客户端防止 XSRF
